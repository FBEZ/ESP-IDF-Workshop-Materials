# Preliminary Knowledge

To follow this workshop, make sure you meet the prerequisites listed below.

## Basic Knowledge

* Basic electronics

  * Resistors, capacitors, DC/DC power supplies
  * Reading a simple electrical schematic

* Basic embedded programming

  * Flash memory
  * Difference between “compiling” and “flashing”
  * Basic knowledge of standard microcontroller peripherals (at least GPIO and I2C)

* Fundamentals of the C language

  * Header files
  * Compiler / linker
  * `define`
  * `struct` and `typedef`

* [JSON](https://en.wikipedia.org/wiki/JSON) and [YAML](https://en.wikipedia.org/wiki/YAML) formats

* HTML and its main tags (`<html>`, `<body>`, `<h1>`, `<h2>`, `<p>`)

* Basic understanding of HTTP request methods (`GET`, `POST`) and the concept of URIs and routes
