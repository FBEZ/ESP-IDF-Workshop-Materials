# Prerequisite Knowledge – Advanced Workshop

To follow this workshop effectively, make sure you meet the prerequisites listed below.

* Good understanding of:

  * Programming in the C language and the linking process
  * Callback functions and function pointers
  * The MQTT protocol and how to use it

* Embedded programming:

  * Flashing/programming and linking operations
  * Familiarity with microcontroller peripherals such as GPIO and I2C
  * Basic experience with ESP-IDF
